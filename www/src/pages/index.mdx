# 🚀 react-rapid-steps
### Effortlessly craft and manage multi-step forms in React!
**Why?:** If you've come up with an idea to create a multi-step form and started asking yourself questions like:
* How can I collect data from multiple steps? 🤔
* How do I manage transitions between the steps in my form? 😐
* Will the user lose all entered data if they refresh the page? 🫣
* What if I want to redirect the user to an external service provider like Auth0 or Stripe? 🫠
* Will releasing a new app version cause crashes if users have old app data in their localStorage? 😵‍💫

#### Keep reading - you are in the right place. We have answers to these questions encapsulated in this small library.
